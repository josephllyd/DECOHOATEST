# DECOHOATEST
Deco Homeowners' Association System Web Application 

How to run Project:

Open VSCODE:
Open terminal
mkdir DecoHOATest
cd DecoHOATest
git clone https://github.com/josephllyd/DECOHOATEST.git
cd DECOHOATEST
cd client
npm install  
cd ..
cd server
npm install
npm start
npm run dev

open new terminal
cd ..
cd client   
DecoHOATest\DECOHOATEST\client  
npm run start
